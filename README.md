# Truck Job Tracker

Application Next.js simple pour suivre les travaux à faire sur des camions.

## Fonctionnalités

- Formulaire avec : numéro d’unité, travail à faire, date et case “Fait”.
- Ajout d’une ligne dans un tableau.
- Modification du statut (coché / décoché) directement dans le tableau.
- Suppression d’une ligne.
- Sauvegarde locale dans `localStorage`.
- Persistance des données après fermeture/réouverture de la page.

## Démarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Aucune variable d’environnement n’est nécessaire.
